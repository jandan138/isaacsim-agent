# Autonomous Robots Migration Notes

This directory is for a journal-style branch aimed at Autonomous Robots if the
paper needs a longer, more system-oriented presentation later.

## Migration goal

Keep the same Block A evidence, but allow:

- fuller system and pipeline description
- more explicit experiment construction details
- stronger discussion of engineering lessons and limitations

Use `paper/outlines/autonomous_robots_outline.md` as the structure guide.

## Best source to branch from

Branch from:

- the neutral mother draft for reusable scientific prose
- the RA-L branch for the most compressed evidence-backed core

## What can stay shared

- definitions of `P0/P1/P2` and `R0/R1`
- metric definitions
- main result statements
- focused ablation statements
- limitations and non-claims, with possible expansion

## What should be expanded or rewritten

- system/pipeline description
- experimental setup detail
- discussion
- limitations section depth
- title and abstract

## Guardrails

- do not widen the science beyond Block A without explicit new evidence
- do not let the longer format turn the paper into a roadmap dump
- keep the same controlled-study identity
